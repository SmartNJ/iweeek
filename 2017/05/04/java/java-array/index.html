二维数组

二维数组，即数组的数组，二维数组的每一个元素都是一个一维数组。

方式一：

数据类型[][] 数组名 = new 数据类型[二维数组的长度/包含以为数组的个数][每个一维数组的长度];
数据类型 数组名[][] = new 数据类型[n][m];
数据类型[] 数组名[] = new 数据类型[n][m];

int[][] arr = new int[3][5]; // 定义了一个基本类型整体的二维数组，包含 3 个一维数组，每个一维数组可以存储 5 个整数。
arr[0] // 第一个一维数组，下标为0
arr[1][3] // 获取第二行第四列上的元素

方式二：

数据类型[][] 数组名 = { {元素},{元素1， 元素2},…… };
